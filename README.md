# Лабораторные работы по Django

Репозиторий содержит лабораторные работы, выполненные в рамках изучения веб-разработки на фреймворке Django.

## Цель работы

Освоение базовых возможностей Django:
- создание проекта и приложений
- настройка маршрутизации
- работа с шаблонами
- подключение статических файлов
- создание моделей и представлений
- реализация форм
- организация регистрации и авторизации пользователей
- добавление клиентской логики на JavaScript

## Состав репозитория

- **lab1** — начальная настройка проекта
- **lab2** — шаблоны и статические ресурсы
- **lab3** — приложение статей
- **lab4** — просмотр отдельной статьи
- **lab5** — создание публикации
- **lab6** — регистрация и вход пользователей
- **lab7** — базовая JavaScript-логика
- **lab8** — расширение интерактивности интерфейса
- **lab9** — итоговая версия проекта

## Технологии
